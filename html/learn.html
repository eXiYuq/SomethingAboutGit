Git相关配置

git config --global user.name 你的用户名

git config --global user.email 你的邮箱地址

git config --global list 查看是否设置成功


文件相关右侧标记

M:已修改(Modified)

A:已添加(Added)

D:已删除(Deleted)

R:已重命名的(Renamed)

C:已复制的(Copied)

U:已更新但未融合的(Updated but Unmerged)


Git基础命令

初始化空的git仓库

在新建的文件夹中右击选择git bash here

git init 此命令用于初始化空的git版本库

记录更新到git仓库

git add 文件名字

git add . 把当前目录下的所有变化文件放入暂存区

git commit -m '提交内容说明'

Git日志及状态查看

git log 查看所有提交的日志记录

git log --oneline 简化查看

git status 查看Git仓库的变化,只能看到未提交的所有变更的文件状态

Git版本回退

git reset --hard 版本号

git log --online 可以查看版本号

git reflog 查看Git所有的操作记录包括reset

clear 清屏
